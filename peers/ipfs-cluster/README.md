# ipfs-cluster

## Features

### Three Applications

- `ipfs-cluster-ctl` - CLI for controlling a cluster
- `ipfs-cluster-service` - Daemon for running a cluster
- `ipfs-cluster-follow` - Daemon for following a cluster
